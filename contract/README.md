# Contracts

This directory defines the benchmark and scoring contracts for the
`frontier_memory` search loop.

## Files

| File | Purpose |
|---|---|
| `SEARCH_CONTRACT.md` | high-level mutation and promotion rules |
| `benchmark_tiers.yaml` | benchmark tier definitions and weights |
| `candidate.template.yaml` | shape of a candidate policy file |
| `scoring.md` | scoring assumptions and guardrails |

These files are used with:

- [../docs/research/program.md](../docs/research/program.md)
- [../candidates/README.md](../candidates/README.md)
- [../logs/README.md](../logs/README.md)
